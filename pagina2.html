🔑 Combinaciones de teclas útiles

Ctrl + C

Copiar — copia el texto que seleccionaste

Ctrl + V

Pegar — pega lo que copiaste

Ctrl + A

Seleccionar todo — marca todo el texto

Ctrl + Z

Deshacer — vuelve atrás el último cambio

✏️ Ejercicios — ¡Copiá el texto exacto!

Ejercicio 1 ⭐ — Mayúsculas y minúsculas

Copiá esta frase exactamente igual, con mayúsculas donde corresponde:

Mi nombre es Milagros.

Ejercicio 2 ⭐⭐ — Números y letras

Escribí esta frase que tiene números:

Tengo 8 años y vivo en Duitama.

Ejercicio 3 ⭐⭐⭐ — La arroba @

Escribí esta dirección de correo. Recordá: la @ se hace con Alt Gr + 2:

milagros@gmail.com

Ejercicio 4 ⭐⭐⭐⭐ — Frase completa

Este es el más difícil. Copiá esta frase con todo: mayúsculas, coma, punto y números:

Hola, tengo 8 años y me llamo Milagros.
